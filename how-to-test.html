How to Test the Colormap Creator

Quick Test Guide

Basic Testing

  1. Open index.html in your browser
  2. Open your browser's developer console (F12 or Ctrl+Shift+J or right-click > Inspect > Console)
  3. Try adding color stops by clicking on the colormap area
  4. Check the console for any error messages

Simple Test Page

If the main application has issues, try the simplified test page to see if the basic functionality works.

Fixing Common Issues

1. Color stops not appearing

This might be due to:

  • JSColor library not loading properly
  • Issues with the rendering process

Check the console for any error messages related to jscolor or canvas rendering.

2. Color sliders not draggable

Common reasons:

  • HTML5 drag and drop API issues in your browser
  • Event listener problems

The application now includes both HTML5 drag-and-drop and a manual drag implementation for better compatibility.

3. Browser Compatibility

The application works best in these browsers:

  • Chrome (latest versions)
  • Firefox (latest versions)
  • Edge (Chromium-based versions)

Files to Check

  • index.html - Main application
  • test.html - Simplified test page
  • js/colormap.js - Core colormap functionality
  • js/app.js - User interface logic
  • css/style.css - Styling

Need More Help?

If you continue experiencing issues:

  1. Try accessing the application from another browser
  2. Check for any JavaScript errors in the console
  3. Verify that all required files are in the correct directory structure
  4. Contact support with details about your issue and any error messages you see
